Applicatie- en mediaontwikkelaar stagiaire

Websiteanalist is een klein bedrijf maar wel een leuk bedrijf. Door hun goede kennis van programmeer talen kun je bijna alles krijgen van een simpele website tot een uitgebreide website met een webshop en alles er bij. Ze denken altijd in oplossingen en proberen een website zo goed mogenlijk naar een klant zijn wensen te maken. Maar als stage loper ben je ook zeker goed bij dit bedrijf ze geven je voldoende tijd om dingen te maken en ze nemen alle tijd voor je om iets uit teleggen als je iets niet snapt.

Werkzaamheden

Bij Websiteanalist krijg je leuke werkzaamheiden die je moet doen op je stage. Je kunt bijvoorbeeld een website moeten aan passen voor een klant, maar het kan ook zijn dat je een website van een klant aan wordpress moet koppelen. Maar bij alles wat je moet doen leer je wel iets van. meestal moet je zelf het uitzoeken hoe het werkt ze leggen je wel dingen uit maar dat zijn in grote lijnen.

Kennis

Als je gaat stage lopen bij dit bedrijf is het wel handig dat je wat kennis hebt van HTML, CSS, PHP en wordpres want daar is het bedrijf veel mee bezig, maar wat extra kennis van CSS want daar moet je veel mee doen bij dit bedrijf.

Functie eisen

  • Kennis van HTML/CSS
  • Basis kennis van wordpress
  • Je beheerst zowel de Nederlandse taal als de Engelse
  • Je kunt goed zelfstandig werken

Aantal uren per week

Het aantal uren wat je per week hebt is 39 uren. Zelf mag kan je je vakantie inplannen met overleg. Gemiddeld heb je 3 weken vakantie.
